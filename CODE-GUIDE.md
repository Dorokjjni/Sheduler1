# Карта проекта

- `index.html` — каркас страницы: заголовок, два SVG-циферблата, форма задачи и недельная таблица.
- `styles.css` — внешний вид, адаптация под телефон, цвета категорий и оформление таблицы.
- `app.js` — создание циферблатов, выбор времени, задачи, даты и сохранение в localStorage.
- `service-worker.js` — сохранение файлов приложения для работы без интернета.
- `manifest.webmanifest` — название PWA, цвета, стартовая страница и иконки.
- `icons/` — иконки приложения.

## Основные данные в app.js

Каждый день хранится примерно так:

```js
{
  focus: "Главная цель дня",
  tasks: [
    {
      id: "...",
      start: 9,
      end: 12,
      title: "Работа над проектом",
      category: "work"
    }
  ]
}
```

`start` включается в промежуток, а `end` является его правой границей. Поэтому задача `9–12` занимает часы 9, 10 и 11.
