# Карта файлов

- `index.html` — видимая структура приложения, два SVG-циферблата и семь столбцов таблицы.
- `styles-v2.css` — оформление в цветах предыдущего приложения.
- `app-v2.js` — выбор часов, создание задач, переключение дней и сохранение в localStorage.
- `service-worker.js` — офлайн-режим и обновление кеша.
- `manifest.webmanifest` — параметры установки PWA.
- `icons/` — иконки приложения.
