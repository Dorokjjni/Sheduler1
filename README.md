# Круговой планировщик — PWA

Приложение повторяет идею бумажного кругового расписания:

- два циферблата: `00:00–12:00` и `12:00–24:00`;
- выбор временного промежутка двумя нажатиями;
- назначение задачи и категории выбранному времени;
- недельная таблица из семи столбцов;
- отдельное поле «Фокус дня»;
- локальное сохранение данных;
- офлайн-режим и установка на домашний экран iPhone.

## Как выбрать время

1. Нажмите сектор часа начала.
2. Нажмите сектор последнего часа.
3. Например, нажатия `9` и `11` создают диапазон `09:00–12:00`.
4. Введите название задачи и нажмите «Сохранить в расписание».

## Запуск на Windows

Откройте терминал в папке проекта и выполните:

```bat
py -m http.server 8080
```

Затем откройте в браузере:

```text
http://localhost:8080
```

## Публикация на GitHub Pages

Загрузите содержимое этой папки в корень репозитория. Файл `index.html` должен находиться на верхнем уровне репозитория.

## Где хранятся данные

Данные сохраняются в `localStorage` браузера. Они не передаются на сервер и не синхронизируются между устройствами.
