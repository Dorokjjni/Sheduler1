# Круговой планировщик — версия 2

Исправленная версия PWA с двумя видимыми циферблатами и недельной таблицей.

## Что исправлено

- Циферблаты теперь записаны прямо в `index.html`, поэтому не исчезают при задержке JavaScript.
- Семь столбцов таблицы также находятся прямо в HTML.
- JavaScript только добавляет выбор времени, задачи и сохранение.
- Файлы получили новые имена `styles-v2.css` и `app-v2.js`, чтобы обойти старый кеш.
- Service Worker использует новый кеш `circle-planner-cache-v2`.
- Дизайн изменён на светло-серый, белый и синий — как у предыдущего приложения.

## Запуск на Windows

```bat
py -m http.server 8080
```

Откройте `http://localhost:8080`.

## Обновление GitHub

Загрузите все файлы из этой папки в корень репозитория и нажмите `Commit changes`.
Старые `styles.css` и `app.js` можно оставить: новая версия их не использует.
После публикации обновите страницу два раза или полностью закройте установленное PWA и откройте снова.
