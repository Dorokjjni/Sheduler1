# Регулярный круговой планировщик — версия 3

PWA-приложение для постоянного недельного расписания. Задачи привязаны к дням недели, а не к календарным датам.

## Как выбрать время

- Одно нажатие на сектор выбирает один час. Например, сектор `6` означает `06:00–07:00`.
- Второе нажатие расширяет промежуток. Например, сначала `6`, затем `8` означает `06:00–09:00`.
- После первого нажатия форма ввода задачи появляется сразу.

## Как работают дни недели

Задача, добавленная в воскресенье, остаётся в воскресном расписании и повторяется каждую неделю. Переключения календарных недель в этой версии нет.

## Запуск на Windows

```bat
py -m http.server 8080
```

Откройте `http://localhost:8080`.

## Основные файлы

- `index.html` — структура циферблатов, формы и семи столбцов.
- `styles-v3.css` — внешний вид и исправление лишнего прямоугольника фокуса SVG.
- `app-v3.js` — выбор времени и хранение регулярных задач.
- `service-worker.js` — офлайн-кеш версии 3.
- `manifest.webmanifest` — настройки установки PWA.
